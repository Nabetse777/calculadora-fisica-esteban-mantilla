# Calculadora de Magnitudes Físicas

**Nombre:** Esteban Mantilla Rubio  
**Materia:** Construcción de Algoritmos  
**Programa:** Ingeniería de Software

## Descripción

Este proyecto es una calculadora web para resolver 10 fórmulas físicas fundamentales usando HTML, CSS y JavaScript.

Incluye:

- Velocidad (v = d / t)
- Aceleración (a = Δv / Δt)
- Fuerza (F = m · a)
- Trabajo (W = F · d · cos(θ))
- Energía Cinética (K = ½ · m · v²)
- Energía Potencial Gravitatoria (U = m · g · h)
- Densidad (ρ = m / V)
- Presión (P = F / A)
- Carga Eléctrica (q = I · t)
- Ley de Ohm (V = I · R)

## Instrucciones para ejecutar

1. Abre el archivo `index.html` en un navegador.
2. Introduce los datos requeridos en cada sección.
3. Pulsa el botón de calcular y obtendrás el resultado.
